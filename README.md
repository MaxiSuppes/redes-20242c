### Prerequisitos
- Python 3.8 o superior
- pip
- virtualenv
- mininet

### Instalación
Crear un virtualenv con python3 y activarlo:

```bash
python3 -m venv venv
source venv/bin/activate
```

Instalar las dependencias:

```bash
pip install -r requirements.txt
```

### Levantar la red
```bash
sudo python3 network.py
```
